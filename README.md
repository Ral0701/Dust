# Repo description | 仓库概述

这是一个集**科技**与**魔法**以及**自动化**的整合包  

## Feature | 特性(~~不是bug~~)

需要玩家在没有泥土的环境中寻找在地底的木材以及物资并创造泥土  
地表只有没用的尘土以及燃烧的灰烬土，地下有一些稀有的木材以及需要提取的木炭
